# Victoria Catroppa - Link Hub

Mini portfolio personale per raccogliere i link principali di Victoria Catroppa, graphic designer.

## Cosa contiene

- Collegamenti a Instagram, Behance e LinkedIn
- Call to action per richieste di collaborazione
- Tema chiaro/scuro con preferenza salvata
- Meta tag social per anteprime più curate
- Logo PNG in homepage e favicon con V nera
- Layout responsive, pulito e accessibile

## Focus

Identità visive, contenuti social e progetti editoriali con un'estetica bold, curata e riconoscibile.
